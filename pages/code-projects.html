pipe7 pipe7 is a PHP library for data processing. It focused on (and optimized for) memory efficiency. It aims to be an alternative to PHP's native array functions (like array_map) and provide a consistent API. In addition to the native functionality, pipe7 also allows you to process iterators without having to convert them to arrays first. GitHub Docs LGPL-3.0-or-later
LibrePages LibrePages is a light PHP framework, which enables easy MVC-like development of REST APIs. It provides all basic blocks which are necessary: routing, request validation, middleware processing, and testing. Simplicity is one large goal of LibrePages, as it does not only enable applications to be developed easier, but it also illustrates, what components are key and build the core of basically any server side framework.
